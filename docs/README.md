# Documentation

Project documentation lives here. The repo root keeps only the files tools
and GitHub expect (`README.md`, `LICENSE`, `CHANGELOG.md`, `AGENTS.md`).

## Guides

| Document                                             | Description                                                                   |
| ---------------------------------------------------- | ----------------------------------------------------------------------------- |
| [Architecture & Design Decisions](./ARCHITECTURE.md) | Demo flow, stack overview, design trade-offs, validation commands, scale path |
| [Architecture Decision Records](./adr/README.md)     | Immutable ADRs (MADR 4.0.0) for load-bearing technical choices                |

## Layout

```
docs/
  ARCHITECTURE.md   # walkthrough + design decisions
  adr/              # numbered ADRs (0001–0010)
```

For agent and contributor conventions, see [AGENTS.md](../AGENTS.md) at the repo root.
